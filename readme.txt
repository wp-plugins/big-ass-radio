Big Ass Radio widget. 15,000 stations in more than 30 genres.

== Description ==
Big Ass Radio gives your visitors access to 15,000 radio stations, podcasts and more. Works on tablets and smartphones. Super easy to use... Select a Genre, select a station, dance in the street.
= What's new in verson 1.0 of this plugin? =
1. Everything! New features coming soon...

== Installation ==
1. Download and unzip bigassradio.zip 
1. Upload the FOLDER containing `bigassradio.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. To add the radio widget on a sidebar, browse to `Appearance > Widgets` and add 'Big Ass Radio' to desired sidebar.
1. To add the radio widget to any post or page you will probably need another plugin such as the 'Widgets in pages' plugin. Direct support for this will be coming in a later release.
